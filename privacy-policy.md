# Privacy Policy for F1 Race HUD
Last updated: 5.Jan.2026.

## Overview
F1 Race HUD is an open‑source Android application developed and published by CodÆrs.  
The app is available on GitHub and distributed through Google Play.  
This Privacy Policy explains what information the app does — and does not — collect.

Information We Do Not Collect  
The app does not collect, store, share, or process any personal data.  

Specifically, the app does not:
- collect personal information
- track users
- use analytics
- display ads
- require or create user accounts
- access contacts, location, storage, camera, microphone, or any sensitive permissions

## Permissions Used  

The app uses only two Android permissions:
1. INTERNET  
   Used solely to allow the app to communicate with online resources required for its functionality.
   The app does not send any personal data over the internet.
2. ACCESS_NETWORK_STATE  
   Used to check whether the device has an active network connection.
   This permission does not provide access to personal information.
   No other permissions are requested or used.

## Open‑Source  
The entire source code is publicly available on GitHub:
https://github.com/AndroidGameDevelopment/F1RaceHUD  
Anyone can review the code to verify that the app does not collect or transmit personal data.

## Third‑Party Services  
The app does not use any third‑party SDKs, analytics tools, advertising networks, or external services that collect user data.

## Children’s Privacy
The app does not collect personal information from anyone, including children under 13.

## Changes to This Policy
If the app’s functionality changes in a way that affects privacy, this policy will be updated accordingly.
Any updates will be published on the same GitHub page where this policy is hosted.
